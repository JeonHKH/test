## 1. 개요
AI 네컷만화는 세 가지 단계로 구분할 수 있습니다.


* 1 프롬프트생성: 프롬프트 생성 api를 사용하여 사용자가 원하는 만화의 줄거리를 입력하면 줄거리를 프롬프트로 바꾸어 생성합니다.

+ 2 그림생성: 스테이블 디퓨젼 api를 활용하여 사용자가 생성한 프롬프트를 토대로 이미지를 생성합니다.

- 3 웹 표시 및 저장: 생성된 이미지를 홈페이지에 표시하고 저장합니다.
